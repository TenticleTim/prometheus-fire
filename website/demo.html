How to install

From zero to governance in 3 steps

1
Install Thesmos
Add to your project as a dev dependency. Works with any Node.js project — Next.js, Express, React, plain TypeScript.
npm install --save-dev thesmos-governance
2
Initialize governance config
The interactive init wizard detects your framework, picks the right rule packs, and configures CI automatically.
npx thesmos init --interactive
3
Add the GitHub Action
Copy the workflow file to .github/workflows/thesmos.yml. Every PR now gets a health score, inline comments, and SARIF output.
npx thesmos ci:github-security --install
