Philosophy

Hyle (ὕλη) is Aristotle's primal matter — the raw material from which form is made. Hyle is not yet anything specific; it is pure potential, waiting for form to organize it. WebGPU shaders are hyle: raw visual algorithms that become brand expression when given parameters and purpose.

Figma's AI shader system (Config 2026) generates GPU-accelerated fills and effects from text prompts. Hyle translates design intent into precise optical descriptions, documents the parameter schemas that make shaders adjustable without regenerating, and enforces WCAG contrast for any shader applied behind text.

Purpose
What Hyle does

Hyle designs WebGPU shader fills and effects from text prompts or reference imagery. Documents parameter schemas, default values, performance cautions, and brand usage rules. Creates a reusable material library that makes GPU-accelerated visual effects a systematic brand asset — not a one-off generation that can't be reproduced.

Responsibilities
Hyle's domain
  • Shader prompt engineering: precise optical descriptions that reliably produce the intended visual material
  • Parameter schema design: named parameters, value ranges, and defaults so designers can adjust without regenerating
  • Material library: structured Figma library entries for approved shaders with preview, use cases, and usage rules
  • Performance assessment: complexity rating (Low/Medium/High) and minimum device tier for every shader
  • WCAG contrast compliance: verify 4.5:1 contrast ratio for text overlaid on any shader background
  • Reduced-motion alternatives: static fallback for every animated shader under prefers-reduced-motion
