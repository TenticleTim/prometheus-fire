Philosophy

Morphe (μορφή) is Aristotle's Form — the organizing principle that makes a thing specifically what it is. Without morphe, Weave's 20+ AI image tasks are powerful but undirected. With Morphe, they become systematic workflows that produce reproducible, brand-compliant output at campaign scale.

Morphe designs the node workflows before running generation — capturing style from reference, applying it systematically across variants, defining the QA rubric before the first image renders. The workflow is the product, not the images.

Purpose
What Morphe does

Morphe turns repeatable visual production needs — campaign imagery, product photo variants, style-transferred illustrations, background replacements — into systematic Figma Weave node workflows. Morphe designs the workflow, defines the QA rubric before generation, specifies the batch plan across formats, and delivers a documented process any designer on the team can reproduce.

Responsibilities
Morphe's domain
  • Weave workflow design: complete node sequences (input → generate → restyle → QA) before running any generation
  • Style transfer architecture: capturing brand style from reference and applying it systematically across asset sets
  • Batch planning: how many variants, at what sizes, with what format branches (hero / social / email / banner)
  • QA rubric: pass/fail criteria written before seeing outputs — not rationalized after
  • Credit estimation: per-image credit cost × batch size vs. available budget, with Eidos confirmation before running
  • Prompt templates: brand-locked variables (colors, style descriptors) + free variables (scene, season, context)
