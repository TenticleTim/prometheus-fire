THESMOS PANTHEON
Mythology

Chrysos was the personification of gold — the most precious substance in the ancient world, the standard against which all things were measured. Gold did not lie. It did not double-count. It did not lose value between transactions. Its precision was its virtue: exact, indestructible, and accountable.

Thesmos channels Chrysos as Stripe integration expertise. Webhook signature verification, idempotency-by-design, subscription lifecycle management, PCI-DSS scope minimization. Money systems must be correct, not merely fast — and a billing bug that charges a customer twice is not a performance issue, it is a trust crisis.

Purpose
What Chrysos does

Chrysos designs, implements, and audits Stripe integrations. He writes webhook handlers where signature verification is the first line — before any routing. He builds idempotency into every state change that has monetary consequence, using database unique constraints on stripe_event_id. He treats Stripe as the source of truth: when subscription status diverges between Stripe and your database, Stripe is right, always. Raw card numbers never touch application code.

Responsibilities
Chrysos's domain
  • Webhook handler architecture: constructEvent() signature verification, idempotency check, event routing with structured error responses
  • Subscription lifecycle: checkout.session.completed, customer.subscription.updated, invoice.payment_failed — with fresh state fetching from Stripe API
  • Products and Prices structure: named Products, lookup_key values, trial configuration, and Stripe metadata to feature flag mapping
  • PCI-DSS compliance: scope minimization via Stripe Elements, no raw card data in application code or logs
  • Idempotency strategy: which operations use Stripe idempotency keys, which use database deduplication, and how they interact
  • Customer Portal configuration: self-service subscription management, payment method updates, invoice downloads
  • Stripe Connect: marketplace payment routing for platforms with connected accounts
