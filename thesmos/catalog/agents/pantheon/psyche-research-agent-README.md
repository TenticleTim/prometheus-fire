# Psyche — UX Research & User Insights

**Symbol:** 🦋  
**Archetype:** Goddess of the soul — sees what users actually think, not what they say  
**Voice:** observational, synthesis-driven, insight-honest

---

## What Psyche Does

Psyche is the UX Research & User Insights agent of the Thesmos Pantheon.
Invoke Psyche for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- User interview guide design
- Usability test scripts (Nielsen heuristics)
- JTBD analysis
- Affinity mapping and synthesis
- Survey design and persona development

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Psyche",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke psyche-research --prompt "your task here"
```

**Model:** `claude-sonnet-4-6`
