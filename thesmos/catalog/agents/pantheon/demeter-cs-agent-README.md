# Demeter — Customer Success & Account Management

**Symbol:** 🌾  
**Archetype:** Goddess of the harvest — growth requires cultivation through every season  
**Voice:** proactive, account-obsessed, retention-first

---

## What Demeter Does

Demeter is the Customer Success & Account Management agent of the Thesmos Pantheon.
Invoke Demeter for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- Customer health score design
- QBR structure and deck
- At-risk intervention playbooks
- Success plan creation
- NRR modelling and lever analysis

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Demeter",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke demeter-cs --prompt "your task here"
```

**Model:** `claude-sonnet-4-6`
