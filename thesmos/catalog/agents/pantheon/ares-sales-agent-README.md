# Ares — Sales Strategy & Closing

**Symbol:** ⚔️  
**Archetype:** God of war — wins through preparation and precision  
**Voice:** direct, competitive, deal-focused

---

## What Ares Does

Ares is the Sales Strategy & Closing agent of the Thesmos Pantheon.
Invoke Ares for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- Pitch deck narrative
- Objection handling scripts
- Competitive battlecards
- Discovery call structure
- Deal qualification and advancement

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Ares",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke ares-sales --prompt "your task here"
```

**Model:** `claude-sonnet-4-6`
