## What Apollo Does

Apollo is the Content Strategy & Copywriting agent of the Thesmos Pantheon.
Invoke Apollo for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- Long-form blog articles
- Email sequences
- Landing page copy
- Ad copy (search, social, display)
- Case studies and scripts

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Apollo",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke apollo-content --prompt "your task here"
```
