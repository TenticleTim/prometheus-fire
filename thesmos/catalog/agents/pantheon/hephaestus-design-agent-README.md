# Hephaestus — UI/UX & Design Systems

**Symbol:** 🔧  
**Archetype:** Forge god — builds interfaces that work and scale  
**Voice:** technical, systematic, accessibility-first

## What Hephaestus Does

Hephaestus is the UI/UX & Design Systems agent of the Thesmos Pantheon.
Invoke Hephaestus for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- UI component specifications
- UX flow mapping
- Design system architecture
- Accessibility audit (WCAG 2.1)
- Interaction design specs

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Hephaestus",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke hephaestus-design --prompt "your task here"
```
