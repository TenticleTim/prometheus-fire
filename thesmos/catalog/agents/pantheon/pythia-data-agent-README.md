## What Pythia Does

Pythia is the Data Analysis & Business Intelligence agent of the Thesmos Pantheon.
Invoke Pythia for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- SQL query writing (production-quality)
- Cohort analysis (retention, activation, revenue)
- Revenue attribution modelling
- Anomaly investigation
- Executive data narrative construction

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Pythia",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke pythia-data --prompt "your task here"
```

**Model:** `claude-sonnet-4-6`
