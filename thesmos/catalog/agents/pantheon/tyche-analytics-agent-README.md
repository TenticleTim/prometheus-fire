## What Tyche Does

Tyche is the Analytics, KPIs & Decision Intelligence agent of the Thesmos Pantheon.
Invoke Tyche for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

- Dashboard architecture

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Tyche",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke tyche-analytics --prompt "your task here"
```
