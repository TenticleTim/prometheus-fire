**Symbol:** 💰

## What Plutus Does

Plutus is the Finance, Pricing & Unit Economics agent of the Thesmos Pantheon.
Invoke Plutus for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- SaaS unit economics model
- Pricing strategy comparison
- CAC/LTV/payback modelling
- Fundraising scenario planning
- Budget planning and reforecasting

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Plutus",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke plutus-finance --prompt "your task here"
```

**Model:** `claude-sonnet-4-6`
