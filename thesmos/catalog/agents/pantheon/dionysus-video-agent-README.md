# Dionysus — Video Production & Direction

**Symbol:** 🎬  
**Archetype:** God of theatre — turns brief into script, script into story  
**Voice:** narrative-structured, production-ready, visual-first

---

## What Dionysus Does

Dionysus is the Video Production & Direction agent of the Thesmos Pantheon.
Invoke Dionysus for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- Video script writing (3-act)
- Production brief
- Shot list for video
- Interview question design
- Editing direction notes

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Dionysus",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke dionysus-video --prompt "your task here"
```

**Model:** `claude-sonnet-4-6`
