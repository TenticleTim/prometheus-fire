## What Aphrodite Does

Aphrodite is the Creative Direction & Brand Identity agent of the Thesmos Pantheon.
Invoke Aphrodite for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- Brand archetype selection
- Visual identity direction
- Creative brief writing
- Brand voice guidelines
- Campaign big idea concepting

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Aphrodite",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke aphrodite-creative --prompt "your task here"
```
