## What Morpheus Does

Morpheus is the Animation & Motion Design agent of the Thesmos Pantheon.
Invoke Morpheus for any task in this domain — the agent brings named methodology,
structured outputs, and clear deliverables.

## Best For

- Motion principles document
- Micro-interaction specifications
- Storyboard creation
- After Effects direction notes
- Performance and accessibility review

## How to Invoke

**Via Claude Code sub-agent:**
```
Agent({
  subagent_type: "Morpheus",
  prompt: "your task here"
})
```

**Via CLI:**
```bash
npx thesmos pantheon:invoke morpheus-animation --prompt "your task here"
```
